## Issues Encountered and Solutions

### Issue 1: Microsoft Defender SmartScreen Warning

When launching the Cursor installer, Windows displayed a SmartScreen warning and prevented the application from running.

**Solution:**
Clicked "More info", verified the publisher as Anysphere, Inc., and selected "Run anyway" to continue the installation.

### Issue 2: Finding the Extensions Marketplace

Initially, I could not locate the Extensions section in Cursor and mistakenly opened local folders instead of the extension marketplace.

**Solution:**
Explored the interface, opened the editor window, and accessed the Extensions marketplace through the View menu. From there I installed the required extensions.
